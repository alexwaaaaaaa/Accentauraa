# Changelog

All notable changes to AccentAura will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Build variants for development, staging, and production environments
- Environment-specific configuration management
- CI/CD pipeline with GitHub Actions
- Automated testing and deployment workflows
- Build scripts for easy variant management
- Version management script
- Comprehensive deployment guide

## [1.0.0] - 2024-01-XX

### Added
- Initial release of AccentAura
- User authentication (email, Google, Facebook)
- 100-level lesson tree with progressive unlocking
- Multiple activity types (flashcards, MCQ, fill-in-blank, listening, speaking)
- AI-powered chat practice with voice input
- AI video interview simulation with confidence analysis
- Gamification features (XP, badges, streaks)
- Leaderboard with rankings
- User profile and progress tracking
- Offline capability with automatic sync
- Firebase Analytics integration
- Firebase Crashlytics for error tracking
- Accessibility features
- Performance optimizations

### Security
- Secure token storage with flutter_secure_storage
- HTTPS for all API communications
- ProGuard rules for Android release builds

## Version Format

- **Major**: Breaking changes or significant new features
- **Minor**: New features, backward compatible
- **Patch**: Bug fixes and minor improvements
